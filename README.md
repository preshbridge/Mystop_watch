# Stopwatch Application:- This is a simple stopwatch application built using C# and Windows Forms.

## Features:
- Start button
- Pause button
- Resume button
- Reset button
- Stop button

## How to Run:
1. Clone the repository
2. Open the solution in Visual Studio.
3. Build and run the project.
